Get Started in 3 Minutes

Installation is quick and painless

Step 1: Install

$ npm install -g agentic-qa

Step 2: Initialize (Optional)

$ agentic-qa init

Step 3: Start Testing

$ agentic-qa scan https://example.com
$ agentic-qa generate-tests
$ agentic-qa compliance-check

Want pre-commit integration?

$ agentic-qa setup-hooks
