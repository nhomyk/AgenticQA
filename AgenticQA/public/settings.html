GitHub Integration

Connect your GitHub repository to enable automated CI/CD pipeline management.

‚ö†Ô∏è Required: You need a GitHub Personal Access Token (PAT) with workflow permissions to use this feature.

Step 1: Create a GitHub Personal Access Token

  1. Go to GitHub ‚Üí Settings ‚Üí Developer Settings ‚Üí Personal Access Tokens ‚Üí Tokens (classic)
  2. Click "Generate new token (classic)"
  3. Give your token a descriptive name (e.g., "AgenticQA Pipeline")
  4. Select the following scopes:
    • repo - Full control of private repositories
    • workflow - Update GitHub Action workflows
    • read:org - Read organization data
  5. Click "Generate token" and copy it immediately
