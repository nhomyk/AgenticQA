### 2. SARIF GitHub Code Scanning ‚Äî Extended to 3 Jobs

**Before:** Only the `sre-agent` CI job uploaded SARIF to GitHub Code Scanning.

**After:** Three jobs now export SARIF findings to GitHub Code Scanning:

| CI Job | SARIF Category | What it covers |
|--------|---------------|----------------|
| `sre-agent` | `agenticqa-sre` | Linting, shellcheck, architectural violations |

### 4. AgenticQA Development Console (client repo: nhomyk/mono)

Created `mono/.github/workflows/agenticqa-console.yml` ‚Äî a comprehensive 870-line
GitHub Actions workflow that turns the "Run workflow" button into a full development
console. No external UI or service required.

**Inputs (9 typed fields):**
- `run_type`: scan | test | feature | full-pipeline | preflight
- `prompt`: natural language feature description
- `test_scope`: unit-and-static | unit-only | static-only | e2e-only | all
- `version_tag`, `priority` (critical/high/medium/low), `ai_mode` (none/suggest/auto)
- `target_branch`, `dry_run`, `requester`

**6 Jobs:**
1. `preflight` ‚Äî PHP + AgenticQA install, git config, `client_preflight` probes
2. `security-scan` ‚Äî `scan_repo()` + SARIF upload (category: `agenticqa-client`)
3. `test-suite` ‚Äî PHPUnit + PHPStan with coverage artifacts
4. `e2e-tests` ‚Äî Playwright on PHP dev server (when scope includes e2e)
5. `feature-pipeline` ‚Äî **Two paths:**
   - `ai_mode=none`: deterministic NLP pipeline (keyword extraction ‚Üí intent classification ‚Üí file impact mapping ‚Üí security consideration detection ‚Üí task breakdown ‚Üí GitHub issue creation) ‚Äî **no API key needed**
   - `ai_mode=auto`: full AgenticQA ADLC via `PromptWorkflowStore`
6. `run-summary` ‚Äî always runs, consolidated results table

Commit: `bf489c6` (nhomyk/mono)

## Quick Start

```bash
# Infrastructure
docker compose -f docker-compose.weaviate.yml up -d

# API (landing page at localhost:8000)
cd AgenticQA && uvicorn agent_api:app --host 0.0.0.0 --port 8000

# Dashboard
streamlit run dashboard/app.py  # ‚Üí localhost:8501

# Demo
python run_demo.py

# Tests
pytest tests/ -m unit -q

# Architecture scan
python run_architecture_scan.py /path/to/repo

# SARIF CLI
python -m agenticqa.export.sarif --sre sre-output.json --mcp mcp-scan-output.json --out results.sarif

# README sync
cp AgenticQA/README.md README.md
git add README.md AgenticQA/README.md
```
