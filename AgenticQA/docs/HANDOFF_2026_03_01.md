## Quick Start

```bash
# Infrastructure
docker compose -f docker-compose.weaviate.yml up -d

# API (landing page at localhost:8000)
cd AgenticQA && uvicorn agent_api:app --host 0.0.0.0 --port 8000

# Dashboard
streamlit run dashboard/app.py  # → localhost:8501

# Demo
python run_demo.py

# Tests
pytest tests/ -m unit -q

# Architecture scan
python run_architecture_scan.py /path/to/repo

# README sync
cp AgenticQA/README.md README.md
git add README.md AgenticQA/README.md
```
