# 0-90 Day Execution Roadmap (ROI OS)

## Day 0-14: Prove value loop
- Finalize portability scorecard APIs and dashboard flows
- Add baseline/delta report export endpoint
- Publish onboarding quickstart for first repository
- Deliver 3 pilot reports with baseline→delta outcomes

### Exit criteria
- At least 3 repos producing weekly delta reports
- One pilot demonstrates measurable CI reliability improvement

## Day 15-45: Operationalize for teams
- Add team/repo portfolio rollups
- Add configurable KPI targets and thresholds
- Add scheduled report generation and notifications
- Improve recommendation quality from observability evidence

### Exit criteria
- At least 2 teams using scorecards weekly
- Repeatable report template accepted by engineering leadership

## Day 46-90: Monetize and expand
- Add controlled remediation workflows (human-in-the-loop)
- Add entitlement gating aligned to Free/Pro/Enterprise
- Add enterprise readiness essentials (RBAC, audit exports baseline)
- Launch sales assets and pricing page

### Exit criteria
- Paid design partner or first contracted customer
- Demonstrated expansion path from scorecard to automation

## Risks and mitigations
- Risk: KPI disputes across teams
  - Mitigation: locked metric dictionary + explicit formulas
- Risk: weak recommendation trust
  - Mitigation: evidence-linked recommendations + confidence score
- Risk: long procurement cycle
  - Mitigation: free local-first pilot with 14-day ROI proof
