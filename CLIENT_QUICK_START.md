# Quick Start - Client Full Pipeline

## In 5 Minutes

### 1. Connect GitHub (1 min)
```
Dashboard → Settings
→ "Connect GitHub" button
→ Paste GitHub PAT token
→ Select your repository
→ Click "Test Connection"
✅ Done
```

### 2. Setup Workflow (1 min)
```
Still in Settings
→ Click "Setup Workflow File"
→ Wait for confirmation
✅ Workflow file created in your repo
```

### 3. Launch Pipeline (30 sec)
```
Dashboard → Main page
→ "Launch Pipeline" button
→ Select: "full" (comprehensive)
→ Select: "main" branch
→ Click "Kickoff Pipeline"
✅ Redirected to GitHub Actions
```

### 4. Watch & Learn (2+ hours)
```
GitHub Actions page
→ See workflow running
→ 12 phases executing
→ Real-time logs
→ Final report when done
✅ Production readiness status
```
